Postgresql 세션 정보확인 및 종료

작성일시: 2022-04-12T23:04:30.000Z
카테고리: Database/Postgresql

PostgreSQL 13.3 버전 기준

현재 접속중인 세션 정보 목록 확인 현재 접속중인 세션의 PID 항목을 복사한다.

SELECT * from pg_stat_activity;

쿼리 실행 취소 쿼리 실행 취소할 세션의 PID를 넣고 쿼리를 실행하면 해당 쿼리는 종료된다.

SELECT pg_cancel_backend(PID);

세션을 종료할려는 경우 아래 쿼리에 PID를 넣고 실행하면, 세션 강제종료된다.

SELECT pg_terminate_backend(PID);
